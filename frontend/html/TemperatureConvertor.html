Aim
Objective of this experiment

To write and understand a C program that converts temperature values bidirectionally between Celsius (°C) and Fahrenheit (°F) scales using standard mathematical conversion formulas, and to visualise the step-by-step computation process through an interactive simulation.

Learning Objective: Develop an intuitive understanding of how the two most commonly used temperature scales — Celsius and Fahrenheit — relate to each other through a linear formula, and how a simple C program can perform both conversions from a single input value.

By the end of this experiment, you will be able to derive the conversion formulas from first principles, trace the arithmetic steps for any given temperature, understand the concept of fixed-point temperature (−40°, where both scales coincide), and apply functions in C to encapsulate reusable conversion logic.

Description
Theory & background

Temperature is a fundamental physical quantity that measures the degree of hotness or coldness of a body. Two scales are universally used in science and daily life — Celsius (°C), proposed by Anders Celsius in 1742, where water freezes at 0° and boils at 100°, and Fahrenheit (°F), proposed by Daniel Fahrenheit in 1724, where water freezes at 32° and boils at 212°.

Both scales are linearly related. The ratio between the two degree-sizes is 9:5 (180 Fahrenheit degrees span the same range as 100 Celsius degrees). The offset of 32 accounts for the difference in the zero point. These two facts together give us both conversion formulas.

The Two Conversion Formulas:

Celsius → Fahrenheit
°F = (°C × 9/5) + 32
Fahrenheit → Celsius
°C = (°F − 32) × 5/9

Notable Reference Points:

Absolute Zero: −273.15°C / −459.67°F
Water Freezes: 0°C / 32°F
Body Temperature: 37°C / 98.6°F
Water Boils: 100°C / 212°F
Equal Point: −40°C = −40°F
Room Temp: 25°C / 77°F

Key Properties of the Conversion:

Relationship: Linear (straight-line)
Degree ratio: 9 °F = 5 °C
Offset: 32°F at 0°C
Data type: float (for precision)
Reversible: F→C→F = original
Space: O(1) — no arrays needed

Procedure
Step-by-step instructions & C program

Follow each step carefully to understand how the temperature conversion program works and how to execute it manually.

  1. Read a single floating-point temperature value from the user using scanf("%f", &temp).
  2. Define function Celsius_to_Fahrenheit(float c) — apply the formula (c × 9 / 5) + 32 and return the result as a float.
  3. Define function Fahrenheit_to_Celsius(float f) — apply the formula (f − 32) × 5 / 9 and return the result as a float.
  4. In main(), call both functions with the same input value to compute c_to_f and f_to_c.
  5. Print both results to 2 decimal places using printf("%.2f\n", ...).
  6. Verify the output: if input is 100 → C→F gives 212.00, F→C gives 37.78.
  7. Use the Simulation tab to interactively trace each arithmetic step for any temperature.
  8. Switch between °C and °F input modes using the dropdown selector in the simulation.
  9. Record results in the Result tab — all conversions performed during the simulation are automatically logged.

Program in C Language:

#include <stdio.h> // Function to convert Celsius to Fahrenheit float Celsius_to_Fahrenheit(float c) { return (c * 9 / 5) + 32; } // Function to convert Fahrenheit to Celsius float Fahrenheit_to_Celsius(float f) { return (f - 32) * 5 / 9; } int main() { float temp, c_to_f, f_to_c; // Input temperature scanf("%f", &temp); // Perform both conversions c_to_f = Celsius_to_Fahrenheit(temp); f_to_c = Fahrenheit_to_Celsius(temp); // Print results printf("Celsius to Fahrenheit: %.2f\n", c_to_f); printf("Fahrenheit to Celsius: %.2f\n", f_to_c); return 0; }
💡 Tip: Note that the program reads one value and performs both conversions simultaneously. This means input 100 is treated as both 100°C (→ 212.00°F) and 100°F (→ 37.78°C) at once. Use the Simulation tab to explore each direction independently.
