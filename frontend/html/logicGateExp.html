Aim
Objective of this experiment

To verify the behavior (truth table) of fundamental digital logic gates — AND, OR, NOT, NAND, NOR, XOR, and XNOR — using integrated circuits and interactive simulation.

Learning Objective: Develop an intuitive understanding of how each logic gate processes binary inputs (0/1) to produce a deterministic output governed by Boolean algebra. Students will observe input-output relationships and verify truth tables for all seven gate types.

By the end of this experiment, you will be able to derive Boolean expressions, verify IC truth tables, trace gate behavior for every input combination, and understand the role of NAND and NOR as universal gates.

Description
Theory & gate reference

A logic gate is a fundamental building block of digital circuits that performs a basic logical function on one or more binary inputs (0 or 1) and produces a single binary output.

In digital electronics, logic 0 ≈ 0V (LOW) and logic 1 ≈ 5V (HIGH) in TTL circuits. Every digital device — from calculators to processors — is built from combinations of these gates. Their behavior is fully described by a truth table.

Key Properties:

Input Type: Binary (0/1)
Output Type: Binary (0/1)
Logic System: Boolean
Gates Covered: 7 Types
Time Complexity: O(1)

Procedure
Step-by-step instructions

Follow each step carefully to verify the truth table of each logic gate.

  1. Navigate to the Simulation tab to access the interactive gate simulator.
  2. Select a logic gate from the selector panel — choose from AND, OR, NOT, NAND, NOR, XOR, or XNOR.
  3. Observe the gate symbol SVG in the visualization area and the Boolean expression displayed below.
  4. Toggle Input A using its switch to set the binary value (0 = LOW, 1 = HIGH).
  5. Toggle Input B (not applicable for NOT gate) to set its binary value.
  6. Observe the output Y — the bulb glows HIGH (💡) or stays dim LOW based on gate logic.
  7. Verify the truth table row highlights to confirm the expected output matches the observed output.
  8. Repeat for all input combinations (00, 01, 10, 11) for each gate to fully verify its truth table.
  9. Switch to each remaining gate and repeat steps 3–8 for all 7 gates.
Tip: The Result tab unlocks automatically once you have interacted with all 7 gates. The progress bar updates as you complete each gate.

Precautions:

  • ● Connect pin 14 to 5V (Vcc) and pin 7 to GND for all TTL ICs.
  • ● Apply logic inputs carefully; ensure correct IC identification before powering on.
  • ● Avoid loose connections on the breadboard to prevent erratic output readings.
  • ● Switch off power supply before inserting or removing ICs from the trainer kit.

Viva Questions
Important viva questions for Logic Gates
1
What is a logic gate?

A logic gate is a basic building block of digital circuits that performs a Boolean operation on one or more binary inputs to produce a single binary output.

2
What are universal gates? Why are they called so?

NAND and NOR gates are called universal gates because any other logic gate — AND, OR, NOT, XOR, etc. — can be implemented using only NAND gates or only NOR gates. This makes them functionally complete.

3
What is the difference between XOR and XNOR?

XOR (Exclusive OR) outputs HIGH (1) only when inputs are DIFFERENT. XNOR (Exclusive NOR) outputs HIGH (1) when inputs are the SAME — it is the complement of XOR and acts as an equality detector.

4
What is the Boolean expression for a NAND gate?

The Boolean expression for a two-input NAND gate is Y = (A · B)̄, which means the output is the complement of the AND operation. The output is LOW only when both A and B are HIGH.

5
What are the IC numbers used for common logic gates?

Common TTL IC numbers: AND → 7408, OR → 7432, NOT → 7404, NAND → 7400, NOR → 7402, XOR → 7486, XNOR → 4077. All TTL ICs require Vcc = 5V on pin 14 and GND on pin 7.
